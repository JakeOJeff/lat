# Syntax Guide
This is a temporary guide on how the language works

## Variable Initialization

### Global Variables

```lua
    x = 10
    x = x + 5
```

### Local Variables

```lua
    nat y = x + 20
    nat z = 20
```

## Functions

```lua
    call funcName(args)
        [body]
    done
```

```lua
    funcName(val)
```

## If Statements

### Normal Statements

```lua
    if (statement)
        [body]
    done
```

### Else and Else If Statements

```python
    if (statement1)
        [body]
    elif (statement2)
        [body]
    else
        [body]
    done
```

## Loops

### While Loops

```ruby
    when (statement)
        [body]
    done
```


## Switch Statements


```java
    switch [value]
       e to [val1]
            [body]
        to [val2]
            [body]
    done

```

## Print Statements

```lua
    print(statement)
```

## Return Statements

```lua
    return [item]
```
